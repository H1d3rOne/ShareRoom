# 网页共享导航按钮设计

## 目标
在网页共享区域增加“后退 / 前进 / 刷新”按钮，仅超级管理员和管理员可用，且操作会全局同步；普通成员仅观看，不可操作。

## 方案
- 在 `pages/room/room.vue` 的网页共享头部加入 3 个按钮。
- 前端维护共享网页历史：`entries[] + index`。
- 新建网页共享、网页内链接跳转、表单跳转时，写入历史。
- 管理员点击后退/前进时，根据历史索引切换 URL，并通过现有 `webpage-share` 事件同步。
- 管理员点击刷新时，重载当前 iframe，不新增历史记录，并保持全局 URL 不变。

## 权限
- 超级管理员 / 管理员：可点击并全局同步。
- 普通成员：按钮禁用。

## 回归验证
- UI 测试覆盖按钮存在、权限禁用态、历史函数、刷新逻辑。
- 运行 `npm test` 与 `npm run build`。
